Este repositorio contiene el material complementario del articulo titulado "Simulador de realidad mixta para cateterismo venoso central: análisis de requerimientos y propuesta conceptual". 

En el archivo "Comparativa de simuladores", se presenta una tabla compárativa de los simuladores para el entrenamiento de punción venosa central.
En el archivo "Lista de pasos generales para CVC", se presenta una lista de 23 pasos genrales para el cteterismo venoso central
Finalmente, en el archivo "Matriz de requerimientos para simuladores", presenta una tabla con 32 requerimientos para simuladores de entrenamiento médico para cateterización venosa central con tecnologías inmersvivas 
, al igual que las referencias con los documentos en donde se mencionan dichas características.
